Section I

Project Overview

Pada bulan maret tahun 2020, pandemi Covid-19 mulai menyebar ke tanah air. Pandemi ini merupakan hal baru bagi masyarakat Indonesia, sehingga pemerintah pusat mulai menyadari bahwasannya masyarakat membutuhkan sebuah media/platform terpercaya yang dapat mempublikasi informasi maupun bantuan terkait pandemi ini.
Challenge
Bagaimana membuat sebuah sistem yang dapat memberikan informasi maupun bantuan secara real-time dan dipercaya oleh masyarakat.
Journey
Dari challenge ini, pemerintah Jawa Barat dengan sigap mengambil keputusan untuk white-label-ing aplikasi yang sudah ada.
Goals & Objective
Dari hasil white-label-ing aplikasi tersebut, kami bertujuan untuk merubah secara iteratif aplikasi ini sesuai kebutuhan user.
Section II

Design Process

❖ Project brief
Pada saat pertama kali rilis, aplikasi ini hanya membutuhkan 2 hari untuk rilis ke publik. Dengan keputusan white-label-ing sebuah aplikasi yang sudah ada, saya sebagai UI Designer ditugaskan untuk me-redesign aplikasi yang sudah ada menjadi aplikasi baru dengan kebutuhan fitur yang berbeda. Untuk mencari tahu problem usability di awal, saya memutuskan menggunakan metode Heuristic Evaluation sebelum melakukan riset lebih lanjut.
❖ Release Version 1.0

Pikobar Mobile Version 1.0

Pada saat sebelum rilis, kami sempat melakukan testing dengan metode smoke test selama 1 hari kepada tim internal Jabar Digital Service dan struktural yang berada di lingkungan pemerintah pusat provinsi Jawa Barat. Smoke test ini bertujuan untuk mencari tahu problem dari sisi fungsional maupun usability difase awal sebelum rilis ke publik.
❖ Update Version 1.5

Pikobar Mobile Version 1.5

Selama kurang lebih 2 bulan dari sejak pertama rilis, aplikasi ini mendapatkan minor update terhadap tampilan UI-nya. Perubahan ini dilakukan karena terdapat feedback dan kebutuhan dari user yang semakin meningkat, salah satunya adalah fitur Cek Sebaran, Bantuan Sosial dan Status Kesehatan bagi yang telah mengikuti tes masif pada aplikasi Pikobar Mobile. Dari sisi tampilan UI juga terdapat perubahan mulai dari white-space hingga visibilitas warna, namun masih dalam 1 package aplikasi yang sama.
❖ Update Version 2.0

Pikobar Mobile Version 2.0

Pada versi 2.0, Pikobar mendapatkan major update mulai dari tampilan UI hingga package code base nya sehingga hampir seluruh user interfacenya berubah total. Tujuan dari perubahan design aplikasi ini adalah untuk meningkatkan user engagement dari sisi kemudahan dalam penggunaan aplikasi dan penyesuaian istilah-istilah baru terkait Covid-19 dari Kementrian Kesehatan Republik Indonesia. Selain itu, terdapat banyak fitur baru maupun fitur yang telah banyak melewati tahapan riset agar pengguna tetap merasakan dampak positif dalam menggunakan aplikasi ini.

Section IV

The Outcome

Pada versi terakhir ini Pikobar kembali mendapatkan major improvement dari sisi UI yaitu penggunaan design system. Selain design system, layout dari beberapa halaman mendapatkan revamp setelah kami melakukan beberapa kali interview & testing kepada user Pikobar pada saat research bersama UX Researcher. Hasil dari versi terakhir ini cukup mendapatkan banyak komentar positif dari para pengguna maupun media massa.
